# BlockFin — Participatory Financing Platform for SMEs

A blockchain-based platform connecting SMEs with investors, guarantors, and administrators through smart contracts on the Ethereum network. Built to remove traditional banking barriers for small businesses.

## Tech Stack
- **Smart Contracts:** Solidity, Hardhat, Sepolia Testnet
- **Authentication:** MetaMask, OAuth2, JWT
- **Backend:** ASP.NET Core (Clean Architecture)
- **Frontend:** Angular
- **Database:** SQL Server
- **DevOps:** Docker, GitHub Actions CI/CD

## Key Features
- Asset tokenization as NFTs
- Full loan lifecycle (request → approval → repayment)
- Role-based access for 4 user types (Owner, Investor, Guarantor, Admin)
- 52 unit tests with 100% pass rate
- Full CI/CD pipeline with zero manual deployments

## Status
Live on Ethereum Sepolia Testnet
